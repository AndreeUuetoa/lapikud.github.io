# how to dev

1. Install Ruby

    sudo apt install ruby

2. Install Jekyll

    gem install jekyll bundler
    
3. Clone repo

    git clone https://github.com/Lapikud/lapikud.github.io.git 
    
4. Bootstrap 

    cd lapikud.github.io
    bundle

5. Serve the page

    bundle exec jekyll serve -w
